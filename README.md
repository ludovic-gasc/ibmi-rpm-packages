# IBM i RPM Packages

Community RPM repository for IBM i / IBM Power, hosted on **GitHub Pages**.

| Repository URL | `https://ludovic-gasc.github.io/ibmi-rpm-packages/` |
|---|---|
| GPG public key | `RPM-GPG-KEY-ibmi` |
| Architecture | `ppc64`, `noarch` |

---

## Quick install on IBM i

```bash
# 1. Download the .repo file
curl -o /QOpenSys/etc/yum/repos.d/ibmi-rpm-packages.repo \
     https://ludovic-gasc.github.io/ibmi-rpm-packages/ibmi-rpm-packages.repo

# 2. Import the GPG signing key
curl -o /tmp/RPM-GPG-KEY-ibmi \
     https://ludovic-gasc.github.io/ibmi-rpm-packages/RPM-GPG-KEY-ibmi
rpm --import /tmp/RPM-GPG-KEY-ibmi

# 3. Refresh cache and install
yum makecache
yum install qpdf
```

## Contributing / Adding a package

### Locally (WSL2)

**Prerequisites** — in WSL2 (Ubuntu/Debian):
```bash
sudo apt install rpm gnupg2 createrepo-c git
```

**Load your GPG key ID** (required for signing):
```bash
export GPG_KEY_ID=$(gpg --list-secret-keys --keyid-format=long ludovic.gasc@be.ibm.com \
  | grep "^sec" | awk '{print $2}' | cut -d'/' -f2 | head -1)
echo "GPG_KEY_ID: ${GPG_KEY_ID}"
```

**Add one or more RPMs** (signs, commits and pushes automatically):
```bash
export GPG_KEY_ID=<your-key-id>
./scripts/add-package.sh path/to/my-package.ppc64.rpm
```

The script will:
1. Copy the RPM to `packages/<arch>/`
2. Rebuild `repodata/` with `createrepo_c`
3. Sign `repomd.xml` + export `RPM-GPG-KEY-ibmi` with GPG (locally)
4. Commit and push to `main`
5. GitHub Actions deploys the updated site to GitHub Pages

**Manually rebuild and sign without pushing**:
```bash
export GPG_KEY_ID=<your-key-id>
./scripts/build-repo.sh --sign
```

## Automated CI/CD

The [`publish.yml`](.github/workflows/publish.yml) workflow triggers on every `push` to `main`.

It performs a **single** task: deploy the repository root to **GitHub Pages**.

> **GPG signing is done locally in WSL2** — the private key never leaves your machine.
> No GitHub Secrets are required.

## Repository structure

```
ibmi-rpm-packages/
├── packages/
│   ├── noarch/          ← Architecture-independent RPMs
│   └── ppc64/           ← IBM Power (IBM i) RPMs
├── repodata/            ← Generated metadata (repomd.xml, etc.)
├── scripts/
│   ├── build-repo.sh        ← Generate repodata + GPG signature
│   ├── add-package.sh       ← Add an RPM to the correct subdirectory
│   └── setup-and-publish.sh ← One-shot full initialisation (WSL2)
├── .github/
│   └── workflows/
│       └── publish.yml  ← GitHub Actions CI/CD
├── ibmi-rpm-packages.repo  ← .repo file for yum/dnf on IBM i
├── RPM-GPG-KEY-ibmi        ← GPG public key (generated by build-repo.sh)
└── README.md
```
