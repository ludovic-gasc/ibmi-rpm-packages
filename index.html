IBM i RPM Packages

Community RPM repository for IBM i / IBM Power — two separate repositories per OS version


ppc64 noarch IBM i 7.4 IBM i 7.6

Quick install on IBM i

IBM i 7.6

1 — Download and enable the IBM i 7.6 repository

curl -o /QOpenSys/etc/yum/repos.d/ludovic-gasc-ibmi-rpm-packages-76.repo \
     https://ludovic-gasc.github.io/ibmi-rpm-packages/ibmi/7.6/ludovic-gasc-ibmi-rpm-packages-76.repo

2 — Install a package

yum makecache
yum install qpdf

IBM i 7.4

1 — Download and enable the IBM i 7.4 repository

curl -o /QOpenSys/etc/yum/repos.d/ludovic-gasc-ibmi-rpm-packages-74.repo \
